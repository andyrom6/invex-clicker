## Features

### Auto Key
- **Spam mode** — rapidly presses any key at your chosen speed
- **Hold mode** — holds a key down continuously
- **Multi-key** — spam multiple keys at once (e.g. `E, F, Space`)
- **30+ keys supported** — Space, E, F, R, Q, W, A, S, D, numbers, and more
- **Humanize timing** — randomizes intervals so it looks natural

### Auto Click
- **Left or Right click**
- **Click at cursor** or **click at a fixed position** on screen
- **Pick position** — click the Pick button, point where you want, and it locks the coordinates
- **Live cursor tracker** — shows your current mouse X/Y in real time

### General
- **Customizable hotkey** — toggle everything with F1–F10, Insert, Home, or End (default: F6)
- **Always on top** — stays visible over your game
- **Activity log** — see exactly what's happening with timestamps
- **Session stats** — tracks total presses, clicks, and session time
- **Settings saved** — your config persists between sessions
- **Speed control** — 1 to 20 presses/clicks per second

## FAQ

**Is it safe?**
It uses Windows `SendInput` with hardware scan codes — the same method your keyboard uses. No injections, no memory editing.

**Will I get banned?**
Use at your own risk. Any automation tool can potentially violate a game's ToS.

**Windows Defender flags it?**
This is a false positive common with PyInstaller `.exe` files. You can click "More info" → "Run anyway", or add an exception.

**Does it work in the background?**
No — it sends input to whatever window is focused. Make sure Roblox is in the foreground.
